Variables and Data Types

Variables are used to store data in a program. Data types define the type of data that a variable can hold. Common data types include:

  • Integer: Whole numbers (e.g., 1, 2, 3)
  • Float: Decimal numbers (e.g., 3.14, 0.001)
  • String: A sequence of characters (e.g., "Hello, World!")
  • Boolean: Represents true or false values
  • Array: A collection of elements (e.g., [1, 2, 3], ["apple", "banana", "cherry"])
  • Object: A collection of key-value pairs (e.g., {name: "Alice", age: 30})

Data Types

Data types can be categorized into primitive and non-primitive types. Primitive data types include integers, floats, strings, and booleans. Non-primitive data types include arrays and objects.

  • Primitive Data Types: Integer, Float, String, Boolean
  • Non-Primitive Data Types: Array, Object
