JavaScript Arrays

An array is an ordered list of values. Each slot has a numeric index starting at 0. Arrays can hold any mix of types (numbers, strings, objects, and so on). Many methods either change the array in place (mutable) or return a new array without changing the original (immutable).

Creating Arrays

  • Array literal: const fruits = ["apple", "banana", "cherry"];
  • Array constructor: const numbers = new Array(1, 2, 3);

Getting Elements

  • By index: fruits[0] returns the first element (e.g. "apple").
  • With .at(): fruits.at(-1) counts from the end (last element).

Adding Elements

  • push(value): Appends to the end. Mutable — changes the original array.
  • unshift(value): Inserts at the start. Mutable.

Removing Elements

  • pop(): Removes and returns the last element. Mutable.
  • shift(): Removes and returns the first element. Mutable.
  • splice(index, count): Removes (or replaces) elements at a given index. Mutable.

Copying and Cloning

  • slice(): const copy = fruits.slice() — shallow copy; original unchanged.
  • Spread: const copy = [...fruits] — new array; original unchanged.

Mutable vs Immutable Patterns

  • Mutable (in place): push, pop, shift, unshift, splice — they modify the same array reference.
  • Immutable-style (new array): slice, concat, spread [...arr] — return a new array; the original is left as it was.

Determining Elements

  • indexOf(value): Returns the index of the first occurrence of a value, or -1 if not found.
  • includes(value): Returns true if the array contains the value, false otherwise.
// Deletermining Array

Determining if a Value is an Array

  • Array.isArray(value): Returns true if the value is an array, false otherwise.
//Nested Arrays

Nested Arrays

Arrays can contain other arrays, creating a nested structure. We can access elements in nested arrays using multiple indices.

  • Accessing Nested Elements: const matrix = [[1, 2], [3, 4]]; Access the number 3 with matrix[1][0].
  • Multi-dimensional Arrays: We can create arrays with more than two levels of nesting, such as const cube = [[[1], [2]], [[3], [4]]]; Access the number 4 with cube[1][1][0].
// Array Destructuring

Array Destructuring

Array destructuring allows us to unpack values from arrays into distinct variables.

  • Basic Destructuring: const [a, b] = [1, 2]; assigns 1 to a and 2 to b.
  • Default Values: const [a, b = 10] = [1]; assigns 1 to a and 10 to b (since the second value is missing).
  • Skipping Values: const [a, , c] = [1, 2, 3]; assigns 1 to a and 3 to c, skipping the second value.
  • Rest: const [a, ...rest] = [1, 2, 3, 4]; assigns 1 to a and [2, 3, 4] to rest.
  • Spread: const arr1 = [1, 2]; const arr2 = [...arr1, 3, 4]; creates arr2 as [1, 2, 3, 4].
  • Swapping: [a, b] = [b, a]; swaps the values of a and b.
// Swapping

Swapping Variables

We can swap the values of two variables using destructuring assignment.

  • Swapping: [a, b] = [b, a]; swaps the values of a and b.
// Merging Arrays

Merging Arrays

We can merge two arrays using the spread operator.

  • Merging: const merged = [...arr1, ...arr2]; creates a new array that combines arr1 and arr2.
// Length of an Array

Length of an Array

The length property of an array returns the number of elements in the array. It is a read-only property that is automatically updated when elements are added or removed from the array. The length of an array is always one more than the highest index in the array. If you set the length property to a value less than the current length, it will truncate the array and remove elements from the end. If you set it to a value greater than the current length, it will create new empty slots at the end of the array.

  • Length: const arr = [1, 2, 3]; console.log(arr.length); outputs 3.
  • Truncating: arr.length = 2; changes arr to [1, 2].
  • Extending: arr.length = 5; changes arr to [1, 2, , , ].
// Array Limit

Array Limit

2 ** 32 - 1 is the maximum length of an array in JavaScript. This is because the length property of an array is a 32-bit unsigned integer, which can represent values from 0 to 2^32 - 1. If you try to set the length of an array to a value greater than this, it will throw a RangeError.

  • Maximum Length: const arr = new Array(2 ** 32 - 1); creates an array with the maximum length.
  • Exceeding Limit: arr.length = 2 ** 32; throws a RangeError.
